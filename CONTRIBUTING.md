Contributions are tolerated but not welcome.

You are free to contribute to add support for your own favorite models but changes to the code logic that affect other models actively being developed by the developer (mostly Qwen family) will likely be rejected.

If you are confident in your code you may still contribute, though consider starting your own fork instead.

# ISSUES ARE VERY WELCOME

Instead of contributing with a PR, you can open an issue.

Discussing possible performance improvements is also strongly encouraged.
